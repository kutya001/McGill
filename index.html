Генерация через ИИ
1. Скопируйте промпт и отправьте в ChatGPT / Claude / Gemini.
2. Сохраните вывод как my_workout.json в папку workouts/.
3. Или нажмите «Импортировать» выше.
Я разрабатываю веб-приложение — фитнес-таймер. Сгенерируй новую программу тренировки строго в формате JSON. Требования к структуре JSON: Корневой объект должен содержать массив "programs", внутри которого находится один объект новой тренировки. Объект тренировки должен содержать поля: id, name, description, estimatedTime (строка, например "15 минут"), exercisesDetails (объект с описанием биомеханики каждого упражнения) и protocol (массив шагов таймера). Правила для exercisesDetails: Ключ — название упражнения. Значение — объект с полями: - goal: Цель упражнения (строка). - start: Исходное положение (строка). - move: Описание движения (строка). *Обязательно включи упражнения "Переход" и "Финиш".* Правила для массива protocol: Каждый элемент — объект шага таймера. Поля: - ex: Название упражнения (совпадает с ключом из exercisesDetails). - phase: Название фазы. - type: Строго одно из: work, rest, prep, finish. - time: Длительность в секундах (число). - voiceTxt: Текст для TTS. Коротко. - toneType: Строго одно из: prep, work, rest, tick, finish. - stepDesc: Текстовое описание шага. Особенности: Всегда начинай с шага "Подготовка" (type: prep, time: 10). Между сменой упражнений вставляй шаг с ex: "Переход" и type: "rest". Завершай массив шагом с ex: "Финиш" (type: finish, toneType: finish). Задание: Сгенерируй программу тренировки: [ВВЕДИТЕ ОПИСАНИЕ ТРЕНИРОВКИ]. Выведи только валидный JSON без markdown-разметки и дополнительного текста.
