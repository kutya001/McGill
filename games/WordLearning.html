Словарь

Учи слова на разных языках
с помощью Google Таблиц

Создаст листы languages, categories, words, users, ratings если их нет

1. Создайте Google Таблицу с 3 листами:
  • languages — столбцы: code, name
  • categories — столбцы: id, name, icon
  • words — столбцы: category, lang_from, word_from, lang_to, word_to, transcription

2. Первая строка каждого листа — заголовки

3. Пример данных в листе words:
  food | ru | Яблоко | en | Apple | ˈæpəl

4. Откройте console.cloud.google.com
  • Создайте проект → Включите Google Sheets API
  • Создайте Service Account (IAM → Service Accounts)
  • Создайте JSON ключ для сервисного аккаунта

5. Откройте доступ к таблице для email сервисного аккаунта:
  ...@...iam.gserviceaccount.com — Редактор

6. ID таблицы — часть URL:
  docs.google.com/spreadsheets/d/{ID}/edit
